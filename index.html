Blacklist Domains

Welcome!

Here's a list of all the blocklists I'm familiar with, compatible with Pi-hole. You may note that some of these lists are mirrored by myself, due to not originally being in a compatible format. If this concerns you, feel free not to use them.

It's also worth noting that if you are wanting a (relatively) hands-off adblocking experience, you will want to consider not installing any 'suspicious-level' domains. I recommend installing these lists to a file called /etc/pihole/adlists.list, which will need to be created if it does not already exist.

The formatting for adlists.list is the same as adlists.default: Customising Sources for Ad Lists.

Whitelist Suggestions:

The following is a list of useful sites that have been found to be blocked at some point:
  • Google Content:
    • google.com: The entire site (Elad Karakos Hosts)
    • webcache.googleusercontent.com: Google Cache (Elad Karakos Hosts)
    • chrome.google.com: Required for Chrome extensions
    • clients2.google.com: Google apps connectivity (Andy Shorts Hosts)
    • clients3.google.com: Google apps connectivity (Andy Shorts Hosts)
    • clients4.google.com: Google apps connectivity (Andy Shorts Hosts)
    • clients5.google.com: Google apps connectivity (Andy Shorts Hosts)
  • Dropbox Content:
    • dropbox.com: The entire site
    • dl.dropbox.com: Required to download content from Dropbox (Joey Lanes Hosts)
    • dl.dropboxusercontent.com: Required to download content from Dropbox
  • gitter.im: Github chat (Elad Karakos Hosts)
  • msftncsi.com: Windows connectivity tester (Elad Karakos Hosts, Mahakala)
  • www.msftncsi.com: Windows connectivity tester (Elad Karakos Hosts, Mahakala)
  • proxy.novafusion.ea.com: Activation server for Origin by EA Games
  • ax.phobos.apple.com.edgesuite.net: Apple App Store content (Elad Karakos Hosts)
  • www.kickstarter.com: Required to access main site (Elad Karakos Hosts)
  • static.kickstarter.com: Kickstarter content (Elad Karakos Hosts)
