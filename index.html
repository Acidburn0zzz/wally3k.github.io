Blacklist Domains

Hi, I'm WaLLy3K (over at /r/pihole) and I've created this page to help people have a more enjoyable online presence - with less ads, trackers and malicious content. These blocklists have original content (thanks to the painstaking work of each list maintainer), and of course, are compatible with Pi-hole. They may work with other adblocking tools, but your milage may vary!

Before starting, here are some reading points:
  • The changelog for this page is found at Github
  • My other project is Pi-hole Block Page
  • Lists bulleted with a tick have no notable false positives
  • Lists bulleted with a cross block multiple useful sites (e.g: Pi-hole updates, Amazon AWS, Google, Netflix)
  • Lists are installed to /etc/pihole/adlists.list, which must be created first
  • In order to easily write to adlists.list, run sudo chown pi /etc/pihole/adlists.list
  • Formatting adlists.list is the same as adlists.default: Customising Sources for Ad Lists
  • Once you have finished adding lists to adlists.list, make sure to run pihole -g
  • I mirror some of these lists. If this (or the lack of TLS) concerns you, then consider parsing them yourself

Whitelisting Suggestions:

The more you block, the more likely you'll come across a false positive. First, you'll want to check out Commonly Whitelisted Domains over at the Pi-hole forums, then you can see if anything below is relevant to your interests. (Tip: Whitelisting and Blacklisting can be performed on multiple sites seperated by a space)
